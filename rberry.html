Workshop on RASPBERY PI

The Raspberry Pi is a series of credit card sized single-board computers developed in UK by the Raspberry Pi Foundation with the intention of promoting the teaching of basic computer science in schools and third world countries.

About Raspberry Pi

Raspberry Pi is based on the Broadcom BCM2835 system on a chip(SoC),which includes an ARM1176JZF-S 700 MHz processor,VideoCore IV GPU,and was originally shipped with 256 megabytes of RAM ,later upgraded(models B and B+) to 512 MB.The system has Secure Digital(SD)(models A and B) or MicroSD(models A+ and B+) sockets for boot media and persistent storage.
The Raspberry Pi primarily uses Linux-kernel based operating systems. The install manager for the Raspberry Pi is NOOBS. The operating systems included NOOBS are:
Arch Linux ARM
OpenELEC
Pidora
Puppy Linux
RISC OS
Raspbian
